## 3. Implementation materials

Templates, assessments, and implementation methods are **excluded** unless explicitly marked as CC BY 4.0.

This exclusion applies to:

- proprietary assessment instruments
- delivery templates not published in this repository
- implementation tooling and automation

Where materials are explicitly marked CC BY 4.0 in the repository, that marking takes precedence over this general exclusion.
